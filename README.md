hm
==

Hipchat Mathias - ping @mafintosh in Hipchat

```javascript
npm install hm -g

// your hipchat authtoken from settings page
hm init <authtoken>

// sent text message to Mathias
hm pingm <message>

// sent text message to other guy
hm ping <user> <message>

// join the chat room
hm chat <room> 
```
