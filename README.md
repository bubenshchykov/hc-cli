
HipChat cli
==

```javascript
npm install hc-cli -g

// your hipchat authtoken from settings page
hip init <authtoken>

// sent a private message
hip ping <user> <message>

// join the chat room
hip chat <room> 
```

And of course you can get directly to Mathias
```javascript
hip pingm <message>
```
